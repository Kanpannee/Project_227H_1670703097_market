# โครงงาน: ตลาดนัดมือสองสำหรับนักศึกษา
นี่คือโปรเจกต์ Backend สำหรับแอปพลิเคชัน "ตลาดของมือสองสำหรับนักศึกษา" พัฒนาด้วย Java Spring Boot ตามขอบเขตงานที่กำหนด

## 🎯 ปัญหาที่ต้องการแก้ไข (Problem)
นักศึกษาต้องการช่องทางในการซื้อของใช้ราคาประหยัด (เช่น หนังสือ, เฟอร์นิเจอร์) แต่ยังไม่มีช่องทางซื้อขายภายในมหาวิทยาลัยที่ปลอดภัยและน่าเชื่อถือ นอกจากนี้ นักศึกษาที่กำลังจะสำเร็จการศึกษา ก็มีของใช้สภาพดีที่ต้องการขายต่อ เพื่อลดปริมาณของเหลือทิ้ง

## 💡 ขอบเขตของโครงงาน (Solution & Scope)
เราจะพัฒนาแอปพลิเคชันที่จำกัดการใช้งาน **เฉพาะนักศึกษาเท่านั้น** โดยมีขั้นตอนการยืนยันตัวตนผ่าน **อีเมลนักศึกษา** เพื่อสร้างชุมชนการซื้อขายที่น่าเชื่อถือ
ผู้ใช้จะสามารถเพิ่มรายการสินค้าตามหมวดหมู่ (เช่น หนังสือ, เฟอร์นิเจอร์, อุปกรณ์อิเล็กทรอนิกส์) ได้

## ✨ ฟีเจอร์หลัก (Core Features)
* **ระบบยืนยันผู้ซื้อ/ผู้ขาย:** ยืนยันตัวตนในมหาวิทยาลัย (ผ่านอีเมลนักศึกษา)
* **รายการตามหมวดหมู่:** แบ่งสินค้าเป็นหมวดหมู่ชัดเจน (หนังสือ, เฟอร์นิเจอร์, อิเล็กทรอนิกส์, ฯ)
* **การสื่อสารภายในแอป:** ผู้ใช้สามารถส่งข้อความเพื่อพูดคุย และต่อรองราคาได้โดยตรง
* **ความปลอดภัย:** มีฟังก์ชันแนะนำจุดนัดพบที่ปลอดภัยภายในมหาวิทยาลัย
* **ความน่าเชื่อถือ:** มีระบบให้คะแนนสภาพสินค้าหลังการซื้อขาย

## 🛠️ Tech Stack (เทคโนโลยีที่ใช้)
* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, JavaScript
* **Database:** เชื่อมต่อฐานข้อมูลสำหรับจัดเก็บข้อมูลผู้ใช้และสินค้า

## 🛑 ขอบเขตที่ไม่รวม (Out of Scope)
* โปรเจกต์นี้ **ไม่รวม** การพัฒนาระบบชำระเงินออนไลน์จริง

## 📂 โครงสร้างโปรเจกต์ (Spring Boot Structure)
โครงสร้างโปรเจกต์ Spring Boot ที่วางแผนไว้:

└── src
    └── main
        ├── java
        │   └── com/university/secondhandmarket
        │       ├── SecondhandMarketApplication.java  
        │       ├── config       // (SecurityConfig)
        │       ├── controller   // (AuthController, ProductController, MessageController)
        │       ├── model (entity) // (User, Product, Category, Message)
        │       ├── repository   // (UserRepository, ProductRepository)
        │       └── service      // (AuthService, ProductService, MessageService)
        │
        └── resources
            ├── static         // (HTML, CSS, JS files)
            ├── application.properties // (Database connection)
